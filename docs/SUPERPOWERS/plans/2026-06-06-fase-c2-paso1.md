# Fase C.2 · Paso 1 — Plan de implementación

> **Para workers agénticos:** SUB-SKILL REQUERIDA: usar `superpowers:subagent-driven-development` para ejecutar este plan tarea por tarea. Los pasos usan checkbox (`- [ ]`).

**Goal:** Click en una sección del canvas → la resalta con el chrome de selección (overlay) + abre su inspector; los botones subir/bajar/duplicar/borrar operan vía las ops del admin que ya patchean en vivo por C.1. Sin edición inline (eso es Paso 2).

**Architecture:** Chrome dentro del iframe (preview-gated), **fuente única `editorState.selection`** (el iframe refleja vía `set-selection`, solo emite `{select}`/`{section-action}`). Overlay flotante anclado al documento (no muta el nodo de la sección → byte-identidad A3). Dos mensajes nuevos validados estándar G3. Deploy **aditivo/dormido**: storefront primero (inerte sin admin nuevo) → gate → admin.

**Tech Stack:** Astro 5 SSR + Cloudflare Workers (storefront), vanilla JS (admin, IIFE `window.TiendaIA.*`), vitest (storefront), node:test/jsdom (admin tests), Playwright (adversarial + live).

**Spec:** `docs/SUPERPOWERS/specs/2026-06-06-fase-c2-paso1-design.md`.

if (msg.type === 'section-action') {
  const ES = window.TiendaIA.editorState;
  const ACTIONS = { up:1, down:1, duplicate:1, remove:1 };
  if (!ACTIONS[msg.action]) return;
  if (typeof msg.sectionId !== 'string' || !/^sec_[a-z0-9]{4,}$/.test(msg.sectionId)) return;
  const sec = ES.findSection(msg.sectionId);
  if (!sec) return;
  const idx = ES.sections.findIndex(s => s.id === msg.sectionId);
  if (msg.action === 'up') ES.reorderSections(idx, idx - 1);
  else if (msg.action === 'down') ES.reorderSections(idx, idx + 1);
  else if (msg.action === 'duplicate') { const nid = ES.duplicateSection(msg.sectionId); if (nid) pendingSelectAfterPatch(nid); }

function postSelection(sectionId) {
  if (!state.iframe || !state.tenantOrigin) return;
  let label = '';
  if (sectionId) {
    const sec = window.TiendaIA.editorState.findSection(sectionId);
    const def = sec && window.TiendaIA.editorSectionDefs.defs[sec.tipo];
    label = def ? def.label : (sec ? sec.tipo : '');

}
