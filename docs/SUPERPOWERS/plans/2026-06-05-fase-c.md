# FASE C — Plan de implementación

> **Para workers agénticos:** SUB-SKILL REQUERIDA: superpowers:subagent-driven-development. Steps con checkbox `- [ ]`. Spec: `docs/SUPERPOWERS/specs/2026-06-05-fase-c-design.md`.

**Goal:** Preview en vivo por PATCH de fragmento SSR por-sección (C.1) + click-to-edit en 2 pasos (C.2). C.3 ya hecho (hotfix).

**Arquitectura:** Una sola fuente de render (BlockRenderer) y una sola de validate+sanitize (fn compartida) → cero drift. Dos carriles: tema = CSS-var postMessage instantáneo (existe); contenido/estructura = fragmento SSR swappeado en el iframe. Flujo desacoplado (patch del state local ~300ms; save 1.5s en paralelo).

**Tech stack:** Astro 5 SSR + CF Workers (storefront, endpoint, bridge), Deno (EF save), vanilla JS (admin), Zod + sanitize-html (ambos runtimes), Playwright (verificación runtime).

## Task 4: Bridge `section-patch` en el storefront (index.astro) — seguridad Q4.3

**Files:** Modify `apps/storefront/src/pages/index.astro` (script preview `isPreview`)

- [ ] **Step 1:** dentro del listener `message` origin-gated existente, agregar handler `section-patch`. **Origin EXACTO** (`e.origin === ADMIN_ORIGIN`; auditar que `theme`/`select` también sean `===`, endurecer si laxo). **Shape estricto ANTES del DOM**:
  ```js
  if (msg.type === 'section-patch') {
    const OPS = { replace:1, insert:1, remove:1, move:1 };
    if (!OPS[msg.op]) return;
    if (typeof msg.sectionId !== 'string' || !/^sec_[a-z0-9]{4,}$/.test(msg.sectionId)) return;
    const cont = document.querySelector('[data-section-id]')?.parentElement;
    const el = document.querySelector('[data-section-id="' + msg.sectionId + '"]');
    if (msg.op === 'remove') { el && el.remove(); return; }
    if (msg.op === 'move') { if (el && cont && Number.isInteger(msg.toIndex)) cont.insertBefore(el, cont.children[msg.toIndex] || null); return; }
    if (typeof msg.html !== 'string') return;
    if (msg.op === 'replace') { el && (el.outerHTML = msg.html); return; }  // html = NUESTRO SSR sanitizado
    if (msg.op === 'insert' && cont && Number.isInteger(msg.index)) {
      const t = document.createElement('template'); t.innerHTML = msg.html.trim();
      const node = t.content.firstElementChild;
      if (node) cont.insertBefore(node, cont.children[msg.index] || null);
    }
  }
  ```
  **Sin re-sanitizar en browser** [Q4.3] (divergiría del publicado + 3er sanitizer). Defensa = fn compartida (origen del html) + origin/shape.
- [ ] **Step 2:** `npm run build` storefront verde. **Commit** `feat(fase-c): bridge section-patch (origin exacto + shape estricto)`.
